<h1>台中資策會- [嵌入式iOS 人才養成班]小專題 - 動物領養查詢小幫手</h1>
<h3>透過抓取政府網站提供的opendata(json格式)來實作提供使用者查詢及分享</h3>
<br />
<h3>動機:常常看到一些流浪動物被欺負的新聞，加上自己也有養寵物，所以想利用這個APP幫流浪動物出點力，目前已上架至APPstor，希望大家能多重視這個議題</h3>
